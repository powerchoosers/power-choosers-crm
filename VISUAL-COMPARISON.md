## Key Visual Elements

### 1. Icons
- **📈 TrendingUp** - Signal Headline (growth/opportunity)
- **💡 Lightbulb** - Signal Detail (insights/information)
- **💬 MessageSquare** - Talk Track (conversation/action)

### 2. Color Coding

#### Headline Section
```
Background: Subtle gradient (white/3% → white/1%)
Border: White 8% opacity
Text: Pure white, 18px, bold
Icon: Blue (#002FA7)
```

#### Detail Section
```
Background: Same gradient as headline
Border: White 8% opacity
Text: Zinc-200 (light gray), 14px
Bullets: Blue dots (#002FA7)
```

#### Talk Track Section (Special)
```
Background: Blue gradient (#002FA7/10% → #002FA7/5%)
Border: Blue 20% opacity
Left accent: Blue 2px border
Text: Zinc-100 (brighter), 14px, italic
Quote marks: Included in text
```

#### Confidence Badge
```
High:   Green background, green text
Medium: Amber background, amber text
Low:    Red background, red text
```

### 3. Typography

```
Headline:    18px, font-semibold, line-height: 1.75
Detail:      14px, font-normal, line-height: 1.75
Talk Track:  14px, font-normal, italic, line-height: 1.75
Labels:      10px, font-mono, uppercase, tracking-widest
```

### 4. Spacing

```
Section padding:     20px (p-5)
Section gap:         16px (space-y-4)
Line height:         1.75 (leading-7)
Label margin:        12px bottom (mb-3)
Bullet gap:          12px (gap-3)
```

## User Flow

### First Time Viewing
1. User opens account page
2. Sees "No intelligence brief generated yet"
3. Clicks "Refresh" button
4. Loading state: "Researching Mill Forest Dental Group..."
5. Content animates in sequentially
6. User naturally reads from top to bottom
7. Can copy entire brief with one click

### Subsequent Refreshes
1. User clicks "Refresh"
2. Old content fades out
3. Loading state appears
4. New content animates in
5. User sees what changed

### Cooldown State
1. User tries to refresh too soon
2. Button is disabled
3. Tooltip shows: "Try again in 45 minutes"
4. User can still copy existing content
